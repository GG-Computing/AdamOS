# OG-OS
OG-OS is an "external operating system" built completely in Python, similar to Python-OS, it runs layered on top of windows.

### Features
> * File manager
> * Network System
> * Python IDE
> * Browser
> * Task Manager
> * OS Settings
> * Video Player
> * Process Viewer
> * Version Check
